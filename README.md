# Progress Bar Component

## Описание

## Компонент `ProgressBarComponent` предназначен для отображения прогресса выполнения процессов в виде кольца.

## Технические особенности

1. **Независимое API**
   Компонент имеет собственное API для управления состоянием.

2. **Состояния**
   - **value** — числовое значение прогресса (0–100).
     - Начало прогресса соответствует 12 часам.
     - Заполнение движется по часовой стрелке.
     - Ограничение: `0 <= value <= 100` для предотвращения визуальных багов.
   - **Animated** — если включено, дуга прогресса начинает вращаться.
   - **hidden** — скрывает весь блок из области видимости.
   - **UIElements** - принимает html элементы для внутренней логики и анимаций

3. **Адаптивность**  
   Компонент корректно отображается на мобильных устройствах в портретной и ландшафтной ориентации.
4. **Методы API для управления состоянием**
   - setValue(value) - метод принимает number-значение полосы прогресса.
   - setAnimationStatus(status) - метод принимает boolean-значение для включения анимации вращения полосы прогресса.
   - setHiddenStatus(status) - метод принимает boolean-значение для сокрытия блока.
   - setUIElements(fill, valueInput, animateCheckbox, hideCheckbox, container, ringSvg) - метод принимает html элементы от родителя чтобы работать с ними внутри.
     - container - контейнер с содержимым блока, нужен для setHiddenStatus чтобы сокрыть содержимое блока.
     - fill - принимает тэг в котором содержится заливка прогресса.
     - valueInput - принимает тэг инпута чтобы через его содержимое менять прогресс и через UI (можно и через вызов метода).
     - animateCheckbox - принимает тэг чекбокса который отвечает за статус анимации.
     - hideCheckbox - принимает тэг чекбокса который отвечает за статус сокрытия.
     - ringSvg - принимает тэг svg кольца откуда в частности берет атрибут "r" для дальнейшей логики и запуска анимаций.
